# 文献综述生成工作流

基于用户提供的多篇文献，生成符合学术规范的文献综述。

## 触发条件

- 用户要求写综述、文献综述
- 用户要求总结研究现状、研究进展
- 用户说"梳理国内外研究进展"、"写一篇综述"
- 用户要求对某个领域的文献进行系统性梳理

## 工作流程

### Step 1: 逐篇深度阅读并提取关键信息

对每篇文献进行系统性阅读：

1. 查看文档大纲：
```
Action: read_document_outline
Action Input: [doc_id]
```

2. 重点阅读引言、方法、结果、讨论章节：
```
Action: read_document
Action Input: {"doc_id": "[doc_id]", "section": "[章节关键词]"}
```

3. 记录详细的分析笔记：
```
Action: write_note
Action Input: {"title": "[文档名] 综述分析", "content": "研究主题: ...\n理论框架: ...\n研究方法: ...\n核心贡献: ...\n研究局限: ...\n与其他文献的关联: ..."}
```

### Step 2: 识别研究主题和分类

读取所有笔记后，识别文献之间的主题关联：

```
Action: read_note
Action Input: all
```

根据笔记内容，将文献按研究主题进行分类。

### Step 3: 如需补充信息，使用语义检索

如果某些文献的关键信息不够完整：

```
Action: recall
Action Input: [针对性的检索查询]
```

### Step 4: 生成文献综述

```
Action: finish
Action Input: [学术文献综述]
```

## 输出格式要求

### 文献综述模板

#### 一、引言
- 研究领域的背景和重要性
- 综述的目的和范围
- 综述的组织结构说明

#### 二、研究现状
按研究主题分类组织（而非按文献逐篇罗列）：

**2.1 [主题方向 A]**
- 该方向的发展脉络
- 代表性研究及其贡献（引用具体文献名称）
- 主要方法和发现

**2.2 [主题方向 B]**
- 该方向的发展脉络
- 代表性研究及其贡献
- 主要方法和发现

**2.3 [主题方向 C]**
- ...

#### 三、研究方法对比
- 各研究采用的方法论概述
- 方法的优劣势对比
- 方法论的发展趋势

#### 四、研究空白与不足
- 现有研究的共同局限
- 尚未充分探索的问题
- 方法论上的不足

#### 五、未来研究方向
- 基于研究空白提出的建议
- 可能的新方法或新视角
- 跨学科融合的机会

#### 六、结论
- 对研究现状的整体评价
- 核心发现的总结
- 对后续研究的启示

## 国内外研究现状专项指南

当用户特别要求"国内外研究现状"时，按以下结构组织：

#### 国外研究现状
- 按时间线或主题梳理国外研究进展
- 标注代表性研究团队/机构
- 总结国外研究的主要特点和趋势

#### 国内研究现状
- 按时间线或主题梳理国内研究进展
- 标注代表性研究团队/机构
- 总结国内研究的主要特点和趋势

#### 国内外对比
- 研究重点的差异
- 方法论的差异
- 发展阶段的差异

## 注意事项

- 按主题组织，不要逐篇罗列文献
- 每个观点都要标注来源文献名称
- 使用学术化的语言风格
- 识别并明确指出研究空白
- 保持客观中立，区分事实和观点
- 如果文献数量较多，优先深度阅读与综述主题最相关的章节
- 综述的长度应与文献数量和内容深度匹配
